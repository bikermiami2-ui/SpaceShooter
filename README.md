# 🚀 Space Shooter

Браузерная HTML5 аркада в космическом сеттинге. Чистый JS без зависимостей.

## 🎮 Играть онлайн

**[https://bikermiami2-ui.github.io/SpaceShooter/](https://bikermiami2-ui.github.io/SpaceShooter/)**

## Управление

| Действие | ПК | Мобильное |
|----------|-----|-----------|
| Движение | WASD / Стрелки | Тап по левой половине экрана |
| Стрельба | Пробел | Тап по правой половине экрана |

## Особенности

- 3 типа врагов: обычный, быстрый, танк
- Волны с нарастающей сложностью
- Бонусы: двойной выстрел (D), щит (S), лечение (+)
- Система частиц при взрывах
- Процедурные звуки через Web Audio API (без файлов)
- Таблица рекордов в localStorage
- Адаптивный дизайн для ПК и мобильных устройств

## Технологии

HTML5 Canvas + Vanilla JavaScript. Никаких сборщиков, npm или CDN.
